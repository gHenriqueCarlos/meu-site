Política de Privacidade

Saiba como coletamos, usamos e protegemos os seus dados pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

1. Finalidade da Coleta de Dados

A coleta de dados pessoais é essencial para garantir o funcionamento adequado do nosso aplicativo/site. Os dados que coletamos são utilizados para as seguintes finalidades:

  • Autenticação de usuários;
  • Personalização da experiência do usuário;
  • Geolocalização de problemas ou serviços relatados;
  • Comunicação com os usuários para envio de atualizações e notificações importantes.

2. Dados Coletados

Durante o uso do aplicativo/site, as seguintes informações podem ser coletadas:

  • Nome completo
  • Endereço de e-mail
  • Senha de acesso (armazenada de forma criptografada)
  • Foto de perfil (opcional)
  • Localização geográfica no momento do envio de um problema ou serviço
  • Imagens e descrições de problemas ou serviços relatados
  • Comentários inseridos pelos usuários

3. Compartilhamento de Dados

O aplicativo/site não compartilha dados pessoais com terceiros, anunciantes ou outras entidades, salvo quando exigido por lei ou ordem judicial. As informações coletadas são usadas exclusivamente para o funcionamento da plataforma e para melhorar a experiência do usuário.

4. Armazenamento e Segurança

Os dados são armazenados em servidores seguros e protegidos por medidas técnicas adequadas, como criptografia de senhas e controles de acesso restritos. Embora adotemos rigorosas práticas de segurança, não podemos garantir a proteção absoluta contra falhas tecnológicas imprevistas.

5. Direitos dos Usuários

Em conformidade com a LGPD, os usuários têm os seguintes direitos em relação aos seus dados pessoais:

  • Confirmar a existência de tratamento de seus dados pessoais;
  • Acessar, corrigir ou excluir seus dados pessoais;
  • Solicitar a portabilidade dos dados, quando aplicável;
  • Revogar o consentimento dado para o tratamento dos dados pessoais.

Para exercer esses direitos, os usuários podem acessar a seção "Remover Dados" no aplicativo/site ou enviar uma solicitação diretamente para o e-mail fornecido na plataforma.

6. Isenção de Vínculo Institucional

O aplicativo/site é uma plataforma independente e não possui vínculo institucional com governos, prefeituras, câmaras municipais ou qualquer entidade pública. A responsabilidade pelas informações disponibilizadas na plataforma é exclusivamente dos usuários.

7. Atualizações desta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir alterações nas práticas de coleta e uso de dados. Recomendamos que os usuários revisem esta página regularmente para se manterem informados sobre qualquer alteração.

8. Contato

Caso tenha dúvidas ou solicitações sobre o tratamento dos seus dados pessoais, entre em contato conosco por meio do e-mail indicado na plataforma.
